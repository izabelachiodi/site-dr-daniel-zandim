# Site Dr. Daniel Zandim — versão 2

Versão visual aproximada da referência enviada, com estética de landing page médica: cores verde-azuladas, seções curvas, cartões arredondados, hierarquia mais comercial e chamadas de agendamento.

Antes de publicar: validar textos médicos; inserir CRM/RQE; confirmar endereço, convênios e horários; substituir os recortes por fotos originais em alta resolução; adicionar Política de Privacidade/LGPD; revisar publicidade médica aplicável.


## Paleta desta versão

Estrutura visual da segunda proposta, com o azul-marinho da primeira entrega e o dourado/amarelo preservado.


## Fotografias pendentes
